Fake News Detection Website

A web-based Fake News Detection for Twitter

GitHub Link Here


It is a website that scans a tweet for a fake news. It is has a chrome extension as a shortcut.

This was our system for our undergraduate thesis titled "Fake News Detection for Twitter using Machine Learning". It was composed using Python and Bootstrap. The system was connected on a Twitter API. The machine learning algorithm is connected on a Flask web server to run on web.

    Software used to create this:
  • Google Chrome
  • VS Code
  • Jupyter Notebook
